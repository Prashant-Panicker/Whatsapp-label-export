# Privacy Policy — WhatsApp Label Exporter

## Overview

WhatsApp Label Exporter is a Chrome browser extension designed to help users export contact information from labels within WhatsApp Web. This extension operates entirely within the user's browser and does not transmit or store personal data on external servers.

## Data Collection

The extension does **not collect, store, or transmit any personal or sensitive user data**.

All processing happens locally inside the user's browser.

The extension only reads information from the currently open WhatsApp Web page when the user explicitly initiates the export process.

## Data Usage

The extension accesses label and contact information displayed in WhatsApp Web in order to generate an export file for the user.

This data is used **only for the purpose of generating a downloadable export file** requested by the user.

## Data Storage

The extension **does not store any user data** outside the browser.

Exported data is saved directly to the user's device through the browser's download functionality.

## Third-Party Services

This extension **does not send data to any external services or third-party servers**.

No analytics, tracking scripts, or remote code are used.

## Permissions Usage

The extension requests certain Chrome permissions strictly to provide its functionality:

* **Host permissions (web.whatsapp.com)**
  Used to read label and contact information from the WhatsApp Web interface selected by the user.

* **Downloads permission**
  Used to allow users to download the exported contact list to their local device.

* **Tabs permission**
  Used to detect and interact with the active WhatsApp Web tab.

These permissions are used solely for the core functionality of the extension.

## User Control

Users can uninstall the extension at any time through Chrome's extensions management page.

Once uninstalled, the extension no longer has access to any browser data.
